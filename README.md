# Aerospace Defect Detector

AI-powered visual inspection system for classifying surface defects 
in aerospace and defense manufacturing components.

## Overview
This project uses transfer learning (ResNet-18) fine-tuned on the 
NEU Surface Defect Dataset to classify 6 types of metal surface 
defects with high accuracy. Built using PyTorch and deployed as 
an interactive web demo via Gradio.

## Status

- PyTorch + torchvision
- Gradio
